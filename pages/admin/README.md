# 👑 Main Admin Dashboard

This directory contains the Super Admin interface for the Alumni Management Portal.

## 🎯 Role Overview
The **Principal Admin** has absolute system oversight and control over all departments, users, and platform content.

## 📂 File Structure
- `dashboard.html` — Global overview with system-wide statistics (12.5k alumni, 6 depts).
- `departments.html` — Manage all departments and assign coordinators.
- `users.html` — Master list of all users with activation toggles.
- `mentors.html` — Manage and re-assign teachers to departments and batches.
- `alumni.html` — Global, filterable database of all alumni.
- `jobs.html` — Final approval step for all alumni job postings.
- `top-alumni.html` — Approve or reject featured alumni suggestions.
- `events.html` — Global event creation and management.
- `chat.html` — Hierarchical chat to contact coordinators and mentors.
- `notifications.html` — Global announcement composer.
- `settings.html` — System-wide security, modules, and platform configuration.
